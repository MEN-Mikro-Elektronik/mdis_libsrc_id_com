ID Library Documentation

This is the documentation of the ID library. The library will be used to access the EEPROM to get the identification of the device / module.

This library contains of:

  • MICROWIRE_PORT functions: MCRW_PORT_Init()
  • ID EEPROM read/write funcitons: m_mread(), m_mwrite(), m_getmodinfo(), m_write(), m_read()
  • USM EEPROM read/write functions: usm_mread(), usm_mwrite(), usm_read(), usm_write()
